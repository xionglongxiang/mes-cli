# MES统一研发脚手架

## About

MES统一研发脚手架

## Getting Started

安装：

```bash
npm install -g @mes-cli/core
# OR
yarn global add @mes/core
```

创建项目

```bash
mes-cli init 
```

发布项目

```bash
mes-cli publish
```

## More

清空缓存：

```bash
mes-cli clean
```

DEBUG 模式：

```bash
mes-cli --debug
```

指定本地包：

```bash
mes-cli init --packagePath /Users/sam/Desktop/mes-cli/packages/init/
```
